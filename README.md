# MERN Authentication Project

A Full Stack MERN Authentication System built using React, Node.js, Express and MongoDB.

## Features

* User Registration
* User Login
* JWT Authentication
* Email Verification
* Forgot Password
* Reset Password
* Profile Management
* Profile Image Upload
* Change Password
* Delete Account
* Admin Dashboard
* Delete Users
* Pagination
*  MongoDB Atlas
*  React Context API
*  Protected Routes

## Tech Stack

### Frontend

* React.js
* React Router DOM
* Axios
* React Toastify

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* Nodemailer
* Multer

## Installation

### Backend

```bash
cd server
npm install
npm start
```

### Frontend

```bash
cd client
npm install
npm run dev
```
